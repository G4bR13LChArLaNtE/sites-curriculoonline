Linguagens de Programação


C: Conhecimento intermediário, utilizado na prototipagem eletrônica com microcontroladores PIC.

C#: Conhecimento básico, utilizado para criação de interfaces de comunicação em projetos eletrônicos.

Kotlin: Conhecimento básico, aplicado na criação de aplicativos Android.

Python: Conhecimento intermediário, com foco em desenvolvimento Web (Flask, Django), APIs REST (FastAPI), banco de dados (SQLAlchemy), automação de processos (Automation Anywhere 360, Celery, Redis), testes automatizados (pytest, unittest, Selenium, Behave), análise de dados (NumPy, Pandas), interfaces (Tkinter, Flet) e criptografia (Bcrypt).

Java: Conhecimento intermediário, voltado para aplicações Web com Spring Boot.

JavaScript: Conhecimento intermediário, utilizado no desenvolvimento de aplicações Web com React (JS e TS).


Outras Linguagens


HTML: Conhecimento intermediário, utilizado em conjunto com frameworks como Bootstrap.

CSS: Conhecimento intermediário, com foco em responsividade (media queries).

SQL: Conhecimento intermediário, voltado para criação e manipulação de bancos de dados relacionais.


Ferramentas


MySQL, PostgreSQL, SQLite, SQL Server 2019, MongoDB: Conhecimento intermediário em modelagem e gerenciamento de bancos de dados.

Photoshop 2021: Conhecimento básico em edição e modificação de imagens.

AWS (EC2, CloudFront, S3): Conhecimento básico para configuração de ambientes em nuvem para desenvolvimento.

Git e GitHub: Conhecimento básico para versionamento de código e gerenciamento de repositórios.

Docker e Docker Compose: Conhecimento básico em criação de ambientes com containers para desenvolvimento e testes.

Heroku: Conhecimento básico em deploy de aplicações Web.
